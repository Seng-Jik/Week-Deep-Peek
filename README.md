# 🧠 每周深度一瞥

> 每个周五的 20:00，一个广度拓展、一盒思维燃料、一篇哲学/历史随笔。

每周一位不同的领航员——可能是合成生物学的工程师，可能是中世纪死囚牢房里的哲学家，也可能是某个被遗忘文明的一件日常器物。不深挖，不论文，**只要那个「有意思」的入口**。

---

> ⚠️ **免责声明**：本仓库所有内容均由 AI 生成，描述可能与事实相去甚远。
> 本文档仅供娱乐和个人学习参考，请勿作为严谨知识来源。

---

## 结构

每期固定三块：

| 模块 | 定位 | 体量 |
|------|------|------|
| 🗺️ 广度拓展 | 一个陌生领域的浓缩漫游 | 主菜，篇幅最大 |
| 🔥 思维燃料 | 一个值得拎出来反复玩味的概念/框架 | 配菜 |
| 🍰 哲学/历史随笔 | 一个人、一本书、一件事，当甜点 | 小短文 |

## 风格

- **给一对，不给一个**：每个观点配上批评者、对立学派或后继修正
- **有意思 > 有深度**：这是一瞥，不是论文
- **能挂钩已有兴趣更好**：游戏设计/编程/AI/音乐/VRChat 等
