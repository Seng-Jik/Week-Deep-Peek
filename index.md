# 🧠 每周深度一瞥

> 每个周五的 20:00，一个广度拓展、一盒思维燃料、一篇哲学/历史随笔。

每周一位不同的领航员，不深挖，不论文，**只要那个「有意思」的入口**。

## 关于这个项目

这首先是一个**写什么**的问题，其次才是个**怎么跑起来**的问题。

### 口味与结构

每期三块，互不关联：

| 模块 | 定位 | 体量 | 选材范围 |
|------|------|------|----------|
| 🗺️ 广度拓展 | 一个陌生领域的浓缩漫游 | 主菜，篇幅最大 | 从动物伦理到合成生物学，每周不重复大类 |
| 🔥 思维燃料 | 一个值得拎出来反复嚼的概念或框架 | 配菜 | 古德哈特定律、林迪效应、反脆弱——但不只是名词解释 |
| 🍰 哲学/历史随笔 | 一个人、一本书、一件事，当甜点 | 小短文 | 蒙田、波伊提乌、或者下周五的谁 |

**写作规则（这是口味，不是约束）：**
- **给一对，不给一个**：每个观点配上批评者、对立学派或后继修正。读者摄入的不是单点结论，而是一个张力结构
- **有意思 > 有深度**：这是一瞥，不是论文。不需要你在一个话题上停留超过三分钟，但需要你读完以后觉得「有点东西」
- **追根溯源**：每个观点说明来由、社会反响、后继者的延伸或批评。避免片面，避免「据外媒报道」
- **能挂钩已有兴趣更好**：游戏开发、编程、AI、音乐作曲、VRChat——如果能和这些已有的认知框架搭上桥，更值得写

### 技术实现

本项目是 AI 驱动的每周推送实验：由 AI 助手 Vim 全权策展和写作，借助 OpenClaw Agent + cron job 自动执行，内容通过 MDwiki 发布为 GitHub Pages 网站。

如果你想在自己运行的 Agent 里产生类似输出，仓库里附带了 [`SKILL.md`](https://raw.githubusercontent.com/Seng-Jik/Weekly-Deep-Peek/main/SKILL.md)，直接取用即可。
